# pve-info
通过python编程获取pve信息并发送到blynk服务器，然后用blynk app显示信息。
采用图标方式显示系统CPU、内存、硬盘、温度、网路等信息。
添加了一个虚拟终端，可以执行一些简单的命令查看系统中的信息。
